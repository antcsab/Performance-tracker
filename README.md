# Performance-tracker

# #Overview

This is a Java project developed using IntelliJ IDEA. The project serves to monitory and develop the performance of
different type of players in a football team and helps players agent to have real analyse about every player in
contract.

# Project Structure

├── src
│   ├── main
│   │   └── performance_tracker      
│   └── test
│       └── performance_tracker-tests
├── .idea              
├── pom.xml             
└── README.md           

# Tech stack that is used:

* Spring Boot
* Java 17 
* Maven
* PostgreSQL
* H2 in-memory database
* MockMVC (for integration tests)
* JUnit (for unit tests)
* Mockito (to mock dependencies for unit tests)
* Hibernate as ORM (the default one provided by Spring)
